# Employee Management System

A Java OOP project that manages employees and demonstrates inheritance through Manager and Developer classes.

## Features

- Store employee information
- Manage managers and developers
- Display salary details
- Reuse code using inheritance

## OOP Concepts Used

- Classes and Objects
- Inheritance
- Base Class
- Derived Classes
- Constructors

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## How to Run

### Compile

```bash
javac Main.java
```

### Execute

```bash
java Main
```
